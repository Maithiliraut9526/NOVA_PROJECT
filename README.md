

# Surgical AI - NOVA PROJECT
An Augmented Reality (AR) platform for surgical visualization and medical training.

## Features
* **AR Organ Visualization:** View high-fidelity 3D models of the Heart, Brain, and Skeletal System.
* **Secure Authentication:** Firebase-powered login system for medical professionals.
* **Interactive Dashboard:** Real-time data overlay for surgical planning.

## Tech Stack
* **Frontend:** React.js, Tailwind CSS
* **3D/AR:** Google Scene Viewer, GLB Models
* **Backend:** Firebase Auth
* **Icons:** Lucide-React
